Vue
Vuex

Vuex：是一个专为 Vue.js 应用程序开发的状态管理模式
①：vuex解决了组件之间同一状态的共享问题 （解决了不同组件之间的数据共享）
②：组件里面数据的持久化
注：小项目不建议使用vuex

Vuex的使用步骤：
    ①：在src目录下新建一个vuex的文件夹
    ②：vuex文件里面新建一个store.js文件
    ③：安装vuex
        cnpm  install  vuex  --save
    ④：在store.js文件里面引入vue   引入vuex   并且user  vuex
        import  Vue  from  'vue'
        import  Vuex  from  'vuex'
        Vue.use(Vuex)
    ⑤：定义数据
        /*  1.state在vuex中用于存储数据  */
        var  state  =  {
            count:1
        }
    ⑥：定义方法      mutations里面放的是方法，方法主要用于改变state里面的数据
        var  mutations  =  {
            incCount(){
                       ++state.count;
            }
        }
    ⑦：将vuex暴露出去
        const  store  =  new  Vuex.Store({
            state,
            mutations
        })
        export  default  store;
    ⑧：在组件中使用vuex的流程
        （1）：引入store( main.js文件 )         import  store  from  '../vuex/store'
        （2）：注册并触发mutations改变state里面的数据
            export  default{
                data(){
                    return {

                    }
                },
                store,
                methods:{
                    incCount(){
                        this.$store.commit('incCount');         // 触发mutations  改变  state  里面的数据
                    }
                }
            }
         （3）：获取state里面的数据
                this.$store.state.数据；
