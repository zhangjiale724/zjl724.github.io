检查数据类型的方法
发表于 2019-11-11 | 更新于 2019-11-11
前端基础
JavaScript

typeof

​ 检查基本类型或对象类型，但是无法检测具体的对象类型（null,[],{} 返回object）

​ 返回的值为字符串

instanceof

​ 检查具体引用类型，返回布尔值

​ 其他用法：

​ 原型.isPropertyOf(对象) 出现在原型链上返回true

​ 对象.instanceof(构造函数) 出现在原型链上返回true

constructor

​ 构造器

​ 对象（包含原型）.constructor === 构造函数
