让一个元素水垂直居中的方案
发表于 2019-09-27 | 更新于 2019-11-24
前端基础
CSS

绝对定位+负margin：

优点：兼容性好;

缺点:必须知道元素的宽高

绝对定位+margin：auto：

缺点:不支持IE7以下的浏览器,已知宽高

绝对定位+CSS3（transform）：

优点：不需要知道元素的宽高

缺点:：兼容性不好，只支持IE9+的浏览器

css3（display:flex）

优点：代码简单易懂

缺点：PC端兼容性不好
